# Contributing to KAGE OS

Thank you for contributing to KAGE OS!

## Pull Request Workflow

1. Fork the repository and create a feature branch.
2. Follow PEP 8 guidelines and write explicit docstrings and static type annotations.
3. Ensure all automated unit tests pass:
   ```bash
   python3 -m unittest discover -s tests
   ```
4. Submit your pull request with a descriptive summary of changes.
